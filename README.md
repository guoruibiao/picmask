# PicMask - 图片合成工具

一个现代化的图片合成工具，允许用户上传图片，选择蒙版，并将多张图片按照蒙版轮廓智能合成。

## 功能特点

- **直观的界面设计**：左侧图片管理区与右侧预览区分区明确
- **多图片上传**：支持点击上传和拖拽上传多种图片
- **蒙版设置**：可将任意图片设为蒙版，仅显示其轮廓
- **颗粒度控制**：通过滑块调整填充图片的颗粒度大小
- **智能合成**：自动将图片填充到蒙版轮廓中
- **结果下载**：一键下载合成后的图片
- **响应式设计**：适配不同屏幕尺寸

## 技术栈

- HTML5
- JavaScript (原生)
- Tailwind CSS v3
- Font Awesome 图标

## 使用方法

1. **上传图片**：点击左侧上传区域或拖拽图片到该区域
2. **设置蒙版**：在上传的图片中选择一张作为蒙版
3. **调整颗粒度**：使用顶部的滑块调整填充图片的大小
4. **合成图片**：点击"合成图片"按钮开始合成
5. **下载结果**：合成完成后，点击"下载图片"保存结果

## 安装与运行

### 方法1：直接打开

1. 克隆或下载本仓库
2. 直接在浏览器中打开 `index.html` 文件

### 方法2：使用本地服务器

```bash
# 使用 Python
python -m http.server 8000

# 或使用 Node.js (需要安装 http-server)
npx http-server -p 8000
```

然后在浏览器中访问 http://localhost:8000

## 功能实现说明

- **图片上传**：使用FileReader API读取本地图片文件
- **蒙版效果**：通过CSS mask-image属性实现蒙版预览
- **图片合成**：使用Canvas API进行像素级操作和图像合成
- **响应式设计**：使用Tailwind CSS的响应式类实现不同设备的适配
- **动画效果**：添加了悬停、过渡和通知动画提升用户体验
